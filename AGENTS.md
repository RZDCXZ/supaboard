# 项目说明

- 这是一个用于学习 Supabase 的开发中项目，技术栈为 Next.js + Supabase。
- 项目文档存放在 `docs/` 目录。

# 协作要求

- 使用中文沟通，先阅读相关代码和文档，再进行修改。
- 保持改动聚焦且尽量小，不擅自扩展需求或重构无关代码。
- 功能、配置、数据库结构或使用方式发生变化时，同步更新 `docs/` 中的相关文档。
- 涉及 Supabase 时，禁止暴露密钥；数据库表默认考虑 RLS，并通过迁移记录结构变更。
- 完成修改后运行与改动相关的检查或测试；无法验证时明确说明。
- 仅当需求歧义会影响架构、数据或安全时再提问，否则基于现有上下文合理推进。
